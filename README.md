# Task Manager Web App

## 📌 Overview
The **Task Manager Web App** is a simple CRUD application that allows users to sign up, log in, and manage their tasks efficiently. It ensures that each user can view, add, edit, and delete their own tasks, providing a structured UI and secure authentication.

## 🚀 Features
- ✅ **User Authentication**: Sign up and log in securely (JWT/Session-based)
- ✅ **Task Management**: Add, edit, delete, and view tasks in a structured UI
- ✅ **Database Integration**: Stores user details and tasks in a database
- ✅ **Secure API**: Protects task-related operations for authenticated users only
- ✅ **Deployment**: Hosted frontend and backend with cloud database

## 🛠️ Tech Stack
### Frontend:
- React.js / HTML + CSS + JavaScript
- Fetch API / Axios for API calls

### Backend:
- Node.js with Express.js
- JWT / Session-based authentication


### Deployment:
- **Frontend**: Vercel / Netlify
- **Backend**: Render / Railway / EC2
- **Database**: MongoDB
